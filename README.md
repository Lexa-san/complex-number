# Complex numbers calculation

## Requirements

* PHP >= 5.5 

## Instalation

### Composer way

Add code bellow to your composer.json in project:
```json
{
    "require": {
        "Lexa-san/complex-number": "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/Lexa-san/complex-number"
        }
    ]
}
```

Then just run in console:
```sh
composer update --no-dev
```
